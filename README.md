### TradeHub

open this project on Netbeans17 and follow these steps
1. Resolve project files
2. Add a server if it isn't
3. Clean and build the project
4. Run the project

### Technologies used:-
1. Front-End Development:
- HTML
- CSS
- Javascript
- BootStrap

2. Back-End Development:
- Java [JDK-11, Tomcat-9]
- JDBC[hibernate]
- Servlet
- JSP

3. Database:
- MySql
